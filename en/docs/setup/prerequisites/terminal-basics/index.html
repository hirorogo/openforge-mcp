ターミナル（コマンドプロンプト）の使い方

ターミナルとは何か​

ターミナルは、パソコンに文字で指示を出す画面です。

普段パソコンを使うときは、マウスでアイコンをクリックして操作しますよね。ターミナルでは、それと同じことを「文字を入力する」ことで行います。

たとえば、フォルダを開くときは普通ならフォルダのアイコンをダブルクリックしますが、ターミナルでは cd フォルダ名 と入力するとそのフォルダに移動できます。

怖がらなくて大丈夫です

ターミナルは映画のハッカーが使っているような画面に見えますが、実際はとても単純なものです。OpenForge を使うために必要なのは、たった 1 行のコマンドをコピーして貼り付けるだけの操作です。この先で練習していきましょう。

ターミナルの開き方​

Windows の場合​

Windows にはターミナルに相当するアプリがいくつかありますが、以下のどれかを使えば大丈夫です。

方法 1: コマンドプロンプト（最も基本的）​

  1. キーボードの Windows キー を押します（画面の左下にある Windows のマーク）。
  2. 「cmd」と入力します。
  3. 「コマンド プロンプト」というアプリが表示されるので、クリックして開きます。

[Screenshot: スタートメニューで cmd と検索した結果。「コマンド プロンプト」が表示されている]

  1. 黒い画面が開きます。これがコマンドプロンプトです。

[Screenshot: コマンドプロンプトの画面。カーソルが点滅している]

方法 2: PowerShell​

  1. キーボードの Windows キー を押します。
  2. 「powershell」と入力します。
  3. 「Windows PowerShell」というアプリが表示されるので、クリックして開きます。

[Screenshot: スタートメニューで powershell と検索した結果]

  1. 青い画面が開きます。これが PowerShell です。コマンドプロンプトとほぼ同じように使えますが、より多くの機能があります。

[Screenshot: PowerShell の画面]

方法 3: Windows Terminal（おすすめ）​

Windows 11 をお使いの場合は、Windows Terminal が標準でインストールされています。

  1. キーボードの Windows キー を押します。
  2. 「terminal」と入力します。
  3. 「ターミナル」というアプリが表示されるので、クリックして開きます。

[Screenshot: スタートメニューで terminal と検索した結果]

  1. Windows Terminal が開きます。この中で PowerShell やコマンドプロンプトを切り替えて使えます。

[Screenshot: Windows Terminal の画面]

どれを使えばいいの？

迷ったら「コマンドプロンプト」を使ってください。OpenForge の操作で必要なコマンドは、どのターミナルでも同じように使えます。

便利な開き方: フォルダから直接開く​

特定のフォルダでターミナルを開きたい場合、もっと簡単な方法があります。

  1. エクスプローラーで目的のフォルダを開きます。
  2. アドレスバー（上部のフォルダパスが表示されている部分）をクリックします。
  3. 「cmd」と入力して Enter を押します。
  4. そのフォルダの場所でコマンドプロンプトが開きます。

[Screenshot: エクスプローラーのアドレスバーに「cmd」と入力している]

Mac の場合​

  1. 以下のいずれかの方法でターミナルを開きます。

方法 A: Finder から開く

  1. Finder を開きます。
  2. 「アプリケーション」フォルダを開きます。
  3. 「ユーティリティ」フォルダを開きます。
  4. 「ターミナル」をダブルクリックします。

[Screenshot: Finder でユーティリティフォルダ内のターミナルを表示している]

方法 B: Spotlight 検索で開く（おすすめ）

  1. キーボードの Command + Space を押します。
  2. 「terminal」と入力します。
  3. 「ターミナル」が表示されるので、Enter を押すか、クリックして開きます。

[Screenshot: Spotlight 検索で terminal と入力した結果]

  1. 白い画面（または黒い画面）が開きます。これがターミナルです。

[Screenshot: Mac のターミナル画面]

基本的なコマンド​

ターミナルで使える基本的なコマンドを覚えましょう。すべてを暗記する必要はありません。必要なときにこのページに戻ってきて確認すれば大丈夫です。

現在の場所を確認する​

ターミナルには「今いる場所」（カレントディレクトリ）という概念があります。パソコンのフォルダ構造の中で、今どこにいるかを表しています。

Windows（コマンドプロンプト）の場合:

cd

cd だけを入力して Enter を押すと、現在の場所が表示されます。たとえば C:\Users\YourName のように表示されます。

Mac / Windows（PowerShell）の場合:

pwd

pwd は「Print Working Directory」（現在のディレクトリを表示する）の略です。

フォルダを移動する​

cd フォルダ名

cd は「Change Directory」（ディレクトリを変更する）の略です。

具体例:

デスクトップに移動する場合:

cd Desktop

一つ上のフォルダに戻る場合:

cd ..

.. は「一つ上のフォルダ」を意味する特別な記号です。

フルパスを指定して移動する場合:

Windows:

cd C:\Users\YourName\Documents\MyProject

Mac:

cd /Users/YourName/Documents/MyProject
パスの入力を楽にする方法

フォルダ名を途中まで入力して Tab キーを押すと、自動的に残りの文字が補完されます。たとえば「Doc」と入力して Tab を押すと「Documents」と自動的に入力されます。

フォルダの中身を一覧表示する​

Windows（コマンドプロンプト）の場合:

dir

Mac / Windows（PowerShell）の場合:

ls

現在の場所にあるファイルやフォルダの一覧が表示されます。

Node.js のバージョンを確認する​

node -v

Node.js が正しくインストールされていれば、バージョン番号（例: v20.11.0）が表示されます。

npx コマンド​

npx コマンド名

npx は、Node.js に付属しているツールで、パッケージ（プログラムの部品）をダウンロードして実行するためのコマンドです。OpenForge のセットアップでよく使います。

npx と npm の違い
  • npm: パッケージをパソコンにインストールするコマンド
  • npx: パッケージを一時的にダウンロードして実行するコマンド

OpenForge の MCP サーバーを起動するときは npx を使うことが多いです。

ターミナルでのコピーと貼り付け​

ターミナルでのコピーと貼り付けは、普通のアプリとは少し操作が異なる場合があります。

Windows のコマンドプロンプト​

操作やり方
コピーテキストをマウスで選択してから、右クリック（または Ctrl + C）
貼り付け右クリック（または Ctrl + V）
コマンドプロンプトでの注意点

古いバージョンのコマンドプロンプトでは Ctrl + C がコピーではなく「コマンドの中断」として動作します。テキストを選択して右クリックする方法が確実です。

Windows の PowerShell / Windows Terminal​

操作やり方
コピーCtrl + C（テキスト選択中）
貼り付けCtrl + V または 右クリック

Mac のターミナル​

操作やり方
コピーCommand + C
貼り付けCommand + V

よくある質問と対処法​

質問 1: コマンドを打ち間違えたらどうする？​

心配ありません。間違ったコマンドを入力しても、たいていの場合は「そのコマンドは見つかりません」というエラーメッセージが表示されるだけで、パソコンが壊れることはありません。

もう一度正しいコマンドを入力し直してください。

直前のコマンドを呼び出す

キーボードの上矢印キー（ ↑ ）を押すと、直前に入力したコマンドが表示されます。修正して再実行するときに便利です。

質問 2: 「'xxx' は、内部コマンドまたは外部コマンド...として認識されていません」と表示される​

このエラーは、入力したコマンドに対応するプログラムがインストールされていないか、PATH（パソコンがプログラムを探す場所の一覧）に登録されていない場合に表示されます。

  • node コマンドの場合: Node.js がインストールされていないか、PATH が通っていません。Node.js のインストールガイドを確認してください。
  • インストール直後にこのエラーが出る場合: ターミナルを一度閉じて、もう一度開いてから試してください。

質問 3: ターミナルの画面がスクロールしすぎて見づらい​

画面をきれいにするコマンドがあります。

Windows:

cls

Mac:

clear

これで画面がきれいになります（過去のコマンドは消えますが、実行結果には影響ありません）。

質問 4: コマンドの実行が終わらない​

コマンドを実行した後、ずっと処理が続いていて次のコマンドが入力できない場合があります。

  • Ctrl + C を押すと、実行中のコマンドを強制的に停止できます。

実際に練習してみよう​

ここまでの内容を実際に試してみましょう。以下の手順を順番にやってみてください。

  1. ターミナルを開きます。

  2. 現在の場所を確認します。

cd
  1. デスクトップに移動します。
cd Desktop
  1. フォルダの中身を確認します。

Windows:

dir

Mac:

ls
  1. 一つ上のフォルダに戻ります。
cd ..
  1. Node.js がインストールされているか確認します。
node -v
  1. npm がインストールされているか確認します。
npm -v

すべて問題なく実行できれば、ターミナルの基本操作はマスターです。

ここまでできれば十分です

OpenForge を使うために必要なターミナル操作は、ドキュメントに書かれたコマンドをコピーして貼り付けて実行するだけです。コマンドを暗記する必要はまったくありません。困ったときはこのページに戻ってきてください。

まとめ​

ここまでの作業で以下が完了しました。

  • ターミナル（コマンドプロンプト / PowerShell / Terminal）の開き方
  • 基本的なコマンド（cd、dir / ls、node -v、npx）
  • コピーと貼り付けの方法
  • エラーが出たときの対処法

ターミナルは最初は難しく感じるかもしれませんが、実際に使うのはごく限られたコマンドだけです。慣れれば自然に使えるようになります。
