シーンを作ってみよう

基本操作に慣れたら、次は1つの完成したシーンを作ってみよう。今回は「屋上庭園」をテーマに、ステップごとにAIと会話しながら作っていくよ。

完成イメージ​

ビルの屋上にある小さな庭園。コンクリートの床の上に植物が並び、ベンチがあって、夕方の柔らかい光に包まれている -- そんなシーンを目指すよ。

ステップ1: 床を作る​

まずは屋上の床から。

あなた:

10メートル x 10メートルの平らな床を作って。素材はグレーのコンクリートっぽい感じで

AIの動き: 薄いボックス（Plane または薄いCube）が作られて、グレーのマテリアルが適用される。

あなた:

床の端に高さ10センチくらいの縁をつけて、水が流れ落ちないようなイメージで

AIの動き: 床の四辺に沿って、低い壁のようなオブジェクトが追加される。

こまめに確認しよう

ここで一度スクリーンショットを撮っておこう。

あなた:

スクリーンショットを見せて

ステップ2: 壁とフェンスを追加する​

屋上だから、安全のためのフェンスが必要だよね。

あなた:

床の北側と東側にガラスのフェンスを設置して。高さは1.2メートルで、フレームはダークグレーの金属にして

AIの動き: 2辺に沿って、半透明のガラスパネルとメタルフレームのフェンスが作られる。

あなた:

南側と西側には同じ高さの木製のルーバー壁を置いて。目隠しになるような感じで

AIの動き: 残り2辺に木目調のルーバー（すき間のある板の壁）が設置される。

ステップ3: 植物を配置する​

庭園のメインである植物を追加していこう。

あなた:

南西の角に大きめの鉢植えを3つ置いて。高さの違う観葉植物が入っているイメージで

AIの動き: 3つの円筒形の鉢と、それぞれ異なる高さの植物オブジェクトが配置される。

あなた:

床の中央あたりに長方形のプランターを2つ、平行に並べて。低い草花が植わっている感じにして

AIの動き: 長方形のプランターボックスが2つ、中に緑のオブジェクトと共に配置される。

あなた:

フェンス沿いにツタのような植物を這わせて

AIの動き: ガラスフェンスの下部に沿って、ツタを表現するオブジェクトが追加される。

植物の表現について

AIが作る植物は、シンプルな形状の組み合わせになることが多い。もっとリアルな植物が欲しい場合は、アセットストアの植物モデルを使うこともできるよ。ここではまず形を作ることを優先しよう。

ステップ4: ベンチを設置する​

くつろげるスペースも作ろう。

あなた:

プランターの間に2人掛けの木製ベンチを1つ置いて。座面は明るい木の色で、脚はブラックの金属にして

AIの動き: ベンチのオブジェクトが作られて、プランターの間に配置される。座面と脚に異なるマテリアルが適用されるよ。

あなた:

ベンチの横に小さな丸テーブルも追加して。高さは60センチくらいで

AIの動き: ベンチの隣に円形のサイドテーブルが配置される。

ここでスクリーンショットを確認しよう。

あなた:

全体が見える角度でスクリーンショットを撮って

ステップ5: ライティングを設定する​

シーンの雰囲気を決めるライティングは大事なポイント。

あなた:

夕方のような暖かいオレンジ色のディレクショナルライト（太陽光みたいな全体的な光）を設定して。角度は低めで、影が長く伸びるようにして

AIの動き: ディレクショナルライトの色がオレンジ寄りに変わり、角度が調整される。シーン全体が夕暮れの雰囲気になるよ。

あなた:

ベンチの近くに暖色のポイントライト（部分的に照らす光）を1つ追加して。足元を柔らかく照らすような感じで

AIの動き: ベンチ付近に小さなライトが追加されて、局所的に明るくなる。

ライティングのコツ

光は1つだけだと平面的になりがち。メインの光（ディレクショナルライト）と補助の光（ポイントライト）を組み合わせると、立体感のある雰囲気になるよ。

ステップ6: スカイボックスを設定する​

スカイボックスは、シーンの背景となる空のこと。

あなた:

スカイボックスを夕焼け空に設定して。オレンジとピンクのグラデーションで、雲が少しある感じ

AIの動き: シーンの背景が夕焼け空に変わる。ライティングとスカイボックスが合わさって、シーン全体の統一感が出るよ。

あなた:

スクリーンショットを見せて

ここで全体の仕上がりを確認しよう。

AIのフィードバックを活かして調整する​

スクリーンショットを見て気になるところがあれば、AIに調整を頼もう。

あなた:

ベンチの位置がプランターに近すぎる気がする。もう少し離して

AIの動き: ベンチの位置が調整される。

あなた:

全体的に暗すぎるから、ライトの明るさを少し上げて

AIの動き: ライトのIntensity（明るさ）が調整されて、シーンが明るくなる。

あなた:

良い感じ。ベンチに座った人の目線からのスクリーンショットを見せて

AIの動き: カメラがベンチの座面の高さに移動して、座った人が見るであろう景色のスクリーンショットを表示してくれる。

調整は何度でもできる

「ちょっと違うな」と思ったら、遠慮なく何度でも調整を頼んで大丈夫。AIとの対話を繰り返しながら、理想のシーンに近づけていくのがOpenForge MCPの使い方だよ。

完成​

おめでとう。これで屋上庭園のシーンが完成した。振り返ると、こんな流れだったね。

  1. 床 -- シーンの土台を作った
  2. 壁とフェンス -- 空間を区切った
  3. 植物 -- 庭園の主役を配置した
  4. ベンチ -- くつろぎスペースを作った
  5. ライティング -- 雰囲気を演出した
  6. スカイボックス -- 背景を設定した

すべての工程を、AIとの会話だけで進められたはず。次のチュートリアルでは、Blenderを使ってオリジナルのモデルを作る方法を学ぶよ。

セーブを忘れずに

せっかく作ったシーンは、保存しておこう。AIに「セーブして」と言えばOK。詳しくはセーブとロードのチュートリアルを見てね。
