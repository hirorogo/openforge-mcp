こう話しかけるだけ

AIがUnity・Blender・Godotを直接操作します

壊れたビルが並ぶ廃墟の街を作って
Terrain + 建物5棟 + ひび割れ道路 + 夕暮れライティングを配置しました。スクリーンショットを確認します...建物の間隔を調整しました。
Blenderで椅子を作って、Unityの部屋に配置して
Blenderでメッシュ生成 → ポリゴン最適化 → FBXエクスポート → Unityにインポート → 部屋に配置しました。
テストプレイしてバグがないか確認して
PlayModeで実行しました。2件検出: 着地時にNullReferenceException / FPSが座標(100,0,200)付近で25に低下。
セーブして
セーブポイント #12 を作成しました: 「廃墟シーン構築 + 椅子配置 + バグ修正前」

他にはない機能

既存ツールにはない独自の設計

AIが結果を見て自分で直す

スクリーンショットで出来栄えを確認し、問題があれば自律的に修正。指示→確認→修正のループをAIが自動で回します。

BlenderとUnityを1会話で横断

Blenderでモデリング → 最適化 → Unityにインポート → シーンに配置。アプリの切り替え不要。

「セーブして」「戻して」で安心

Gitを知らなくてもバージョン管理。壊れても一言で復元。破壊的な操作の前に自動バックアップ。

AIがゲームを遊んでバグ発見

PlayModeでゲームを実行、入力をシミュレート。エラー・パフォーマンス問題を自動レポート。

テキストから3Dモデルやテクスチャを生成

Rodin, Meshy, Stable Diffusion, DALL-E などと連携。生成→最適化→配置まで自動。

自分のツールを1行で追加

C#なら [OpenForgeTool]、Pythonなら @openforge_tool を付けるだけ。コミュニティでツールを共有できます。
