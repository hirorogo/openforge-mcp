# Using Godot

A reference for common Godot operations, with natural language instruction examples.


## Node operations

| What you want to do | What to say | Tool used |
|---|---|---|
| Add a node | `Add a MeshInstance3D node` | `add_node` |
| Delete a node | `Delete the Player node` | `delete_node` |
| Move a node | `Set the Player's position to (0, 5, 0)` | `set_node_property` |
| View the node list | `Show me the scene tree` | `get_scene_tree` |
| Rename a node | `Rename Node3D to Player` | `rename_node` |


## Scripts

| What you want to do | What to say | Tool used |
|---|---|---|
| Create a GDScript | `Create a script for Player` | `create_script` |
| Read a script | `Show me the contents of player.gd` | `read_script` |
| Edit a script | `Change the movement speed to 10` | `edit_script` |
| Connect a signal | `Connect the button's pressed signal` | `connect_signal` |


## Materials
