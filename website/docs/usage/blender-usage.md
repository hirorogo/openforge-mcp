# Blender での使い方

Blender でよく使う操作を、自然言語での指示例とともにまとめたリファレンスです。


## オブジェクト作成

| やりたいこと | こう言う | 使われるツール |
|---|---|---|
| メッシュを作る | `Cubeを追加して` | `add_mesh` |
| カーブを作る | `ベジェカーブを追加して` | `add_curve` |
| 空オブジェクトを作る | `Emptyを追加して` | `add_empty` |
| テキストを作る | `3Dテキストを追加して` | `add_text` |
| オブジェクトを複製する | `選択中のオブジェクトを複製して` | `duplicate_object` |


## メッシュ編集

| やりたいこと | こう言う | 使われるツール |
|---|---|---|
| 頂点を移動する | `選択した頂点を上に1m動かして` | `edit_mesh` |
| 面を押し出す | `選択した面を押し出して` | `extrude_faces` |
| ループカットを入れる | `X方向にループカットを3本入れて` | `loop_cut` |
| 頂点をマージする | `選択した頂点をマージして` | `merge_vertices` |
| 法線を再計算する | `法線を外向きに揃えて` | `recalculate_normals` |


## マテリアル

| やりたいこと | こう言う | 使われるツール |
|---|---|---|
| マテリアルを作る | `赤いマテリアルを作って` | `create_material` |
| ベースカラーを変える | `マテリアルの色を青にして` | `set_material_property` |
| テクスチャを設定する | `画像テクスチャを設定して` | `set_material_texture` |
| メタリックを変える | `メタリックを1.0にして` | `set_material_property` |
| ノードを接続する | `ノイズテクスチャをバンプに繋いで` | `connect_shader_nodes` |


## モディファイア

| やりたいこと | こう言う | 使われるツール |
|---|---|---|
| Subdivisionを追加する | `サブディビジョンを追加して、レベル2で` | `add_modifier` |
| Mirrorを追加する | `ミラーモディファイアを追加して` | `add_modifier` |
| Booleanを追加する | `ブーリアンで穴を開けて` | `add_modifier` |
| モディファイアを適用する | `サブディビジョンを適用して` | `apply_modifier` |
| Arrayを追加する | `配列モディファイアで5個並べて` | `add_modifier` |


## UV

| やりたいこと | こう言う | 使われるツール |
|---|---|---|
| UV展開する | `スマートUV展開して` | `uv_unwrap` |
| シームをつける | `選択した辺にシームをつけて` | `mark_seam` |
| UVマップを確認する | `UVマップを見せて` | `get_uv_info` |
| UV座標をリセットする | `UVをリセットして` | `reset_uv` |


## アニメーション

| やりたいこと | こう言う | 使われるツール |
|---|---|---|
| キーフレームを打つ | `現在のフレームにキーフレームを打って` | `insert_keyframe` |
| フレーム範囲を設定する | `フレーム範囲を1-120にして` | `set_frame_range` |
| アニメーションを再生する | `アニメーションを再生して` | `play_animation` |
| キーフレームを削除する | `全キーフレームを削除して` | `delete_keyframes` |
| FPSを変更する | `フレームレートを30にして` | `set_fps` |


## レンダリング

| やりたいこと | こう言う | 使われるツール |
|---|---|---|
| レンダリングする | `レンダリングして` | `render_image` |
| エンジンを変更する | `レンダーエンジンをCyclesにして` | `set_render_engine` |
| 解像度を変更する | `解像度を1920x1080にして` | `set_render_resolution` |
| サンプル数を変更する | `サンプル数を256にして` | `set_render_samples` |
| 出力先を設定する | `出力先を /tmp/render にして` | `set_output_path` |


## アーマチュア

| やりたいこと | こう言う | 使われるツール |
|---|---|---|
| アーマチュアを追加する | `アーマチュアを追加して` | `add_armature` |
| ボーンを追加する | `腕のボーンを追加して` | `add_bone` |
| ウェイトペイントする | `自動ウェイトを設定して` | `auto_weight_paint` |

| IKを設定する | `脚にIKを設定して` | `add_ik_constraint` |


## VRM

| やりたいこと | こう言う | 使われるツール |
|---|---|---|

| VRMをエクスポートする | `VRMとして書き出して` | `export_vrm` |
| 表情を設定する | `笑顔のブレンドシェイプを設定して` | `set_vrm_expression` |
| スプリングボーンを設定する | `髪にスプリングボーンを設定して` | `set_vrm_spring_bone` |


## Import / Export

| やりたいこと | こう言う | 使われるツール |
|---|---|---|
| FBXを読み込む | `FBXファイルをインポートして` | `import_fbx` |
