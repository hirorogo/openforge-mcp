# Node.js のインストール

## Node.js とは何か

Node.js は、パソコンの上でさまざまなアプリやツールを動かすための「土台」のようなソフトウェアです。

たとえば、ゲームで遊ぶためにゲーム機が必要なように、OpenForge の MCP サーバーを動かすためには Node.js が必要です。Node.js そのものは画面に何も表示しませんが、裏側でいろいろなプログラムを動かす役割をしています。

:::info なぜ Node.js が必要なの？
OpenForge の MCP サーバーは Node.js の上で動きます。Node.js がインストールされていないと、サーバーを起動できません。最初に一度だけインストールすれば、あとは意識する必要はありません。
:::

## Windows でのインストール

### 手順 1: ダウンロードページを開く

1. ブラウザ（Chrome、Edge など）を開きます。
2. アドレスバー（画面の上のほうにある、URL が表示されている細長い欄）に `https://nodejs.org/ja` と入力して、Enter キーを押します。
3. Node.js の公式サイトが表示されます。

[Screenshot: Node.js 公式サイトのトップページ。緑色のダウンロードボタンが 2 つ表示されている]

### 手順 2: LTS 版をダウンロードする

1. 画面に大きな緑色のボタンが 2 つ表示されています。
2. 左側の「LTS」と書かれたボタンをクリックしてください。
   - LTS は「Long Term Support（長期サポート）」の略で、安定して使えるバージョンという意味です。
   - 右側の「最新版」は新しい機能が入っていますが、不具合が起きやすいため、LTS を選んでください。
3. クリックすると、インストーラー（`.msi` というファイル）のダウンロードが始まります。

[Screenshot: ブラウザの下部にダウンロード中のファイルが表示されている]

:::tip ダウンロード先がわからないとき
ダウンロードしたファイルは通常「ダウンロード」フォルダに保存されます。キーボードの `Windows キー + E` を押してエクスプローラーを開き、左側の「ダウンロード」をクリックすると見つかります。
:::

### 手順 3: インストーラーを実行する

1. ダウンロードした `.msi` ファイルをダブルクリックして開きます。
2. 「このアプリがデバイスに変更を加えることを許可しますか？」という画面が出たら「はい」をクリックします。

[Screenshot: Windows のユーザーアカウント制御ダイアログ]

### 手順 4: インストールウィザードを進める

インストーラーが起動すると、いくつかの画面が順番に表示されます。基本的にはそのまま「Next」を押していけば大丈夫です。

**画面 1: Welcome（ようこそ）**

1. 「Welcome to the Node.js Setup Wizard」と表示されます。
2. 「Next」ボタンをクリックします。

[Screenshot: Node.js インストーラーの Welcome 画面]

**画面 2: License Agreement（ライセンス契約）**

1. ライセンスの内容が表示されます。
2. 「I accept the terms in the License Agreement」（ライセンスに同意する）のチェックボックスにチェックを入れます。
3. 「Next」ボタンをクリックします。

[Screenshot: ライセンス同意画面。チェックボックスの位置を矢印で示している]

**画面 3: Destination Folder（インストール先）**

1. インストール先のフォルダが表示されます。
2. 変更する必要はありません。そのまま「Next」をクリックします。

[Screenshot: インストール先フォルダの選択画面]

**画面 4: Custom Setup（カスタムセットアップ）**

1. インストールする機能の一覧が表示されます。
2. 何も変更せず、そのまま「Next」をクリックします。

[Screenshot: カスタムセットアップ画面]

**画面 5: Tools for Native Modules（ネイティブモジュール用ツール）**

1. 「Automatically install the necessary tools...」というチェックボックスが表示される場合があります。
2. このチェックは入れなくても構いません。OpenForge を使う分には不要です。
3. 「Next」をクリックします。

[Screenshot: ネイティブモジュールツールの画面]

**画面 6: Ready to install（インストール準備完了）**

1. 「Install」ボタンをクリックします。
2. インストールが始まります。数分かかることがあります。

[Screenshot: Install ボタンが表示された画面]

**画面 7: Completed（完了）**

1. 「Finish」ボタンをクリックして、インストーラーを閉じます。

[Screenshot: インストール完了画面]

### 手順 5: インストールを確認する

インストールが正しく完了したかを確認します。

1. キーボードの `Windows キー` を押します（画面の左下にある Windows のマーク）。
2. 「cmd」と入力します。
3. 「コマンド プロンプト」というアプリが表示されるので、クリックして開きます。

[Screenshot: スタートメニューで cmd と検索した結果]

4. 黒い画面が開きます。ここに以下の文字を入力して、Enter キーを押します。

```bash
node -v
```

5. `v20.xx.x` のようなバージョン番号が表示されれば成功です（数字はインストールした時期によって異なります）。

[Screenshot: コマンドプロンプトで node -v を実行し、バージョン番号が表示されている]

6. 続けて、以下も入力して Enter を押します。

```bash
npm -v
```

7. `10.x.x` のようなバージョン番号が表示されれば、npm（Node.js に付属するパッケージ管理ツール）も正しくインストールされています。

:::warning バージョン番号が表示されない場合
「'node' は、内部コマンドまたは外部コマンド、操作可能なプログラムまたはバッチ ファイルとして認識されていません。」というエラーが出た場合は、この後の「よくあるエラーと対処法」を読んでください。
:::

## Mac でのインストール

### 方法 A: 公式サイトからダウンロードする方法

1. ブラウザで `https://nodejs.org/ja` を開きます。
2. 「LTS」と書かれた緑色のボタンをクリックします。
3. `.pkg` ファイルがダウンロードされます。
4. ダウンロードしたファイルをダブルクリックして開きます。
5. 画面の指示に従って「続ける」を何度かクリックし、最後に「インストール」をクリックします。
6. Mac のパスワードを求められたら入力します。
7. インストールが完了したら「閉じる」をクリックします。

[Screenshot: Mac の Node.js インストーラー画面]

**確認方法:**

1. Finder を開きます。
2. 「アプリケーション」 > 「ユーティリティ」 > 「ターミナル」をダブルクリックして開きます。
3. 以下を入力して Enter を押します。

```bash
node -v
```

4. バージョン番号が表示されれば成功です。

### 方法 B: Homebrew を使う方法

:::info Homebrew とは
Homebrew は Mac 用のパッケージ管理ツールです。ターミナルにコマンドを入力するだけでソフトウェアをインストールできます。既に Homebrew を使っている方はこちらの方法が簡単です。Homebrew をまだインストールしていない場合は、方法 A を使ってください。
:::

1. ターミナルを開きます。
2. 以下のコマンドを入力して Enter を押します。

```bash
brew install node
```

3. インストールが完了したら、以下で確認します。

```bash
node -v
```

## よくあるエラーと対処法

### エラー 1: 「node is not recognized」または「node: command not found」

このエラーは、パソコンが Node.js のインストール場所を見つけられないときに発生します。これは「PATH」という設定の問題です。

**Windows での対処法:**

1. パソコンを再起動してみてください。再起動するだけで直ることが多いです。
2. 再起動しても直らない場合は、以下の手順で PATH を手動で設定します。

**PATH を手動で設定する手順（Windows）:**

1. キーボードの `Windows キー` を押して「環境変数」と入力します。
2. 「システム環境変数の編集」をクリックします。

[Screenshot: スタートメニューで「環境変数」を検索した結果]

3. 「環境変数」ボタンをクリックします。

[Screenshot: システムのプロパティ画面。「環境変数」ボタンの位置を示している]

4. 上半分の「ユーザー環境変数」の中から「Path」を探して選択し、「編集」をクリックします。

[Screenshot: 環境変数の一覧画面。Path が選択されている]

5. 「新規」ボタンをクリックします。
6. 以下のパスを入力します。

```
C:\Program Files\nodejs\
```

7. 「OK」をクリックしてすべてのウィンドウを閉じます。
8. コマンドプロンプトを一度閉じて、もう一度開いてから `node -v` を試してください。

[Screenshot: Path の編集画面に Node.js のパスが追加されている]

**Mac での対処法:**

1. ターミナルを閉じて、もう一度開いてみてください。
2. それでも直らない場合は、以下のコマンドを入力します。

```bash
export PATH="/usr/local/bin:$PATH"
```

### エラー 2: インストーラーが起動しない

- ダウンロードしたファイルが壊れている可能性があります。ファイルを削除して、もう一度ダウンロードしてください。
- ウイルス対策ソフトがブロックしている場合があります。一時的にウイルス対策ソフトを無効にしてからインストールを試してみてください。

### エラー 3: 「permission denied」と表示される（Mac）

Mac で `npm install` などを実行したときにこのエラーが出ることがあります。

```bash
sudo npm install -g <パッケージ名>
```

のように、コマンドの先頭に `sudo` をつけてから実行してください。Mac のパスワードを求められたら入力します（入力中、文字は画面に表示されませんが、正常です）。
