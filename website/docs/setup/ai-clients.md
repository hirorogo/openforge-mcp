# AIアプリの設定

OpenForge MCP を使うには、AI と会話するためのアプリが必要です。以下のページで、各アプリごとの詳しいセットアップ手順を確認できます。

---

## どのアプリを選べばいいの?

はじめての方は、この流れで選んでみてください。

- **「とにかく簡単に始めたい」** → [Claude Desktop](/docs/setup/clients/claude-desktop) がおすすめ。設定がいちばん少ないです
- **「プログラミングもやりたい」** → [Cursor](/docs/setup/clients/cursor) または [VS Code](/docs/setup/clients/vscode) がおすすめ。コードを書く機能も充実しています
- **「ターミナルで作業したい」** → [Claude Code CLI](/docs/setup/clients/claude-code) または [Gemini CLI](/docs/setup/clients/gemini-cli) がおすすめ。コマンドラインから AI を使えます
- **「インターネットに繋がない環境で使いたい」または「完全に無料で使いたい」** → [LM Studio](/docs/setup/clients/lm-studio) または [Ollama](/docs/setup/clients/ollama) がおすすめ。自分のパソコンだけで AI を動かせます
- **「どれかわからない」** → まずは [Claude Desktop](/docs/setup/clients/claude-desktop) から始めてみてください

:::info 補足
どのアプリを選んでも、OpenForge MCP でできることは同じです。違いは「AI との会話のしやすさ」や「追加機能」です。あとから別のアプリに切り替えることもできます。
:::

## 各アプリのセットアップガイド

| アプリ | 特徴 | ガイド |
|---|---|---|
| Claude Desktop | デスクトップチャットアプリ。設定が簡単 | [セットアップ手順](/docs/setup/clients/claude-desktop) |
| Cursor | AI搭載コードエディター。開発と3D制作の両立 | [セットアップ手順](/docs/setup/clients/cursor) |
| VS Code (GitHub Copilot) | 定番エディター + Copilot Chat | [セットアップ手順](/docs/setup/clients/vscode) |
| Claude Code CLI | ターミナルベースの AI ツール | [セットアップ手順](/docs/setup/clients/claude-code) |
| Gemini CLI | Google のターミナル AI ツール | [セットアップ手順](/docs/setup/clients/gemini-cli) |
| LM Studio | ローカル AI（GUI付き）。無料、オフライン対応 | [セットアップ手順](/docs/setup/clients/lm-studio) |
| Ollama | ローカル AI（CLI）。無料、オフライン対応 | [セットアップ手順](/docs/setup/clients/ollama) |
