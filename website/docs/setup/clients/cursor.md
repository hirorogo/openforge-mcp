# Cursor

このページでは、Cursor に OpenForge MCP を接続する方法を説明します。

---

## Cursor とは

Cursor は、AI による支援機能が組み込まれたコードエディターです。コードを書きながら AI と会話したり、AI にコードの生成や修正を依頼したりできます。MCP に対応しているため、OpenForge MCP と接続すると、エディター内のチャットから Unity や Blender などの3Dアプリを操作できるようになります。

プログラミングと3D制作の両方をやりたい方に特に向いています。

---

## セットアップ方法

### 方法1: 自動セットアップ（おすすめ）

ターミナルで以下のコマンドを実行するだけで、設定ファイルが自動的に作成されます。

```bash
npx openforge-mcp setup --client cursor
```

「Setup complete」と表示されれば成功です。このあと「動作確認」のセクションに進んでください。

### 方法2: 手動セットアップ

手動で設定する場合は、以下の手順で進めてください。

#### 1. 設定ファイルを作成する

Cursor の MCP 設定ファイルは以下の場所にあります。

```
~/.cursor/mcp.json
```

:::tip ヒント
`~` はホームディレクトリを意味します。Windows の場合は `C:\Users\ユーザー名\.cursor\mcp.json` になります。
:::

#### 2. 設定ファイルを編集する

`mcp.json` をテキストエディターで開き（ファイルがなければ新規作成し）、以下の内容を入力して保存します。

```json
{
  "mcpServers": {
    "openforge-mcp": {
      "command": "npx",
      "args": ["openforge-mcp", "start"]
    }
  }
}
```

:::warning 注意
すでに他の MCP サーバーの設定がある場合は、`"mcpServers"` の中に `"openforge-mcp"` のブロックだけを追記してください。

```json
{
  "mcpServers": {
    "既存のサーバー": {
      "command": "...",
      "args": ["..."]
    },
    "openforge-mcp": {
      "command": "npx",
      "args": ["openforge-mcp", "start"]
    }
  }
}
```
:::

#### 3. Cursor を再起動する

設定を反映するために、Cursor を一度終了してからもう一度起動してください。

## 動作確認

以下の手順で、OpenForge MCP が正しく接続されているか確認します。

1. Cursor を起動します
2. **Ctrl + L**（Mac は **Cmd + L**）でチャットパネルを開きます
3. Composer モードまたは Chat モードで、以下のように入力します

> 「OpenForge MCP のツール一覧を表示して」

ツールの一覧が返ってくれば、接続は成功です。

また、Cursor の設定画面（**Ctrl + Shift + P** → 「Cursor Settings」 → 「MCP」）で、「openforge-mcp」が一覧に表示されていて、ステータスが緑色になっていることでも確認できます。

## 使い方の例

チャットパネルから自然な日本語で話しかけるだけで操作できます。

> 「Cubeを位置 0, 2, 0 に作って」

> 「シーンにあるすべてのオブジェクトを一覧表示して」

> 「点光源を追加して、色を青にして」

Cursor の Composer 機能を使えば、コードの生成とシーンの操作を同時に行うこともできます。たとえば、C# スクリプトを生成してもらいながら、そのスクリプトが動作するシーンを OpenForge MCP で構築してもらう、といった使い方が可能です。

## トラブルシューティング

### MCP の設定画面で接続が赤色になっている

- `~/.cursor/mcp.json` の JSON が正しいか確認してください。よくあるミスはカンマの抜けや括弧の閉じ忘れです
- Node.js がインストールされていることを確認してください（ターミナルで `node --version` を実行して v18 以上であること）
- Cursor を再起動してください

### チャットで MCP ツールが使われない

- チャットパネルのモードが「Agent」または「Composer」になっていることを確認してください。「Chat」モードでは MCP ツールが使えない場合があります
- 明確にお願いしてみてください: 「OpenForge MCP を使って、シーンにキューブを追加して」

### 設定ファイルが見つからない

`~/.cursor/` フォルダがない場合は、Cursor を一度起動してからもう一度確認してください。初回起動時にフォルダが作成されます。フォルダが存在するのにファイルがない場合は、`mcp.json` を新規作成してください。

### npx コマンドが見つからない

Node.js がインストールされていない可能性があります。[nodejs.org](https://nodejs.org) から LTS 版をダウンロードしてインストールしてください。インストール後、ターミナルを再起動してから `npx --version` で確認できます。
